# Детектирование мотоциклов на полосах общественного транспорта

## Описание проекта:
Данный проект разработан в рамках учебной практики и представляет собой
веб-сервис для автоматического обнаружения мотоциклов, движущихся по полосе
общественного транспорта.

Система анализирует изображения дорожной обстановки, определяет наличие
обозначения полосы ОТ (буква «A»), находит границу полосы с помощью белой
дорожной разметки и проверяет, пересекает ли мотоцикл эту границу.
Результаты детектирования сохраняются в базе данных и доступны в виде
статистики и отчётов.

---

## Используемые технологии:
- Python 3.11
- FastAPI
- PostgreSQL
- SQLAlchemy
- YOLOv26 (детекция объектов)
- YOLOv26-seg (сегментация)
- OpenCV
- JavaScript (Frontend)
- HTML / CSS
- ReportLab (PDF-отчёты)
- OpenPyXL (Excel-отчёты)


## Принцип работы системы:
1. Пользователь загружает изображение через веб-интерфейс.
2. Backend выполняет детекцию буквы «A» (обозначение полосы ОТ).
3. Вокруг найденной буквы формируется область интереса.
4. С помощью метода Хафа находится ближайшая белая дорожная разметка —
   граница полосы общественного транспорта.
5. С помощью YOLOv26-seg определяется маска мотоцикла.
6. Проверяется, пересекает ли маска мотоцикла найденную границу.
7. Результат (нарушение / отсутствие нарушения) сохраняется в базе данных.
8. Пользователь может просматривать историю, статистику и скачивать отчёты
   в форматах PDF и Excel.


## Инструкция по запуску проекта:

1. Клонировать репозиторий:
   - «git clone https://github.com/dayerzz/MTUCI_winter_practice.git»
   - «cd MTUCI_winter_practice»

2. Установить зависимости backend:
   - «cd backend»
   - «pip install -r requirements.txt»

3. Создать файл .env и указать строку подключения к базе данных:
   - «DATABASE_URL=postgresql://user:password@localhost:5432/traffic_lane_db»

4. Запустить backend:
   - «python -m uvicorn server:app --reload»

5. Запустить frontend:
   - запустить через Live Server (VS Code) файл index.html

6. Перейти в браузере:
   - Frontend: http://127.0.0.1:5500
   - Backend API: http://127.0.0.1:8000

## Результат работы:
Система корректно определяет факт нарушения движения мотоциклов по полосе
общественного транспорта, сохраняет результаты проверок и предоставляет
удобный интерфейс для анализа статистики и формирования отчётов.
